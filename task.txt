Для всех задач данные следует генерировать с помощью библиотеки "Extention.hpp"

Задание #1
Напишите программу которая сортирует массив из структур Student. (N = 100)
struct Student
{
  int Id;
  int Grade;
  string Name;  
};
Критерий сортировки необходимо спрашивать у пользователя. (по Id, по Grade, или по Name)
Id в диапазоне от 1 до 1000, Grade от 0 до 100, Name состоит от 3 до 6 английских символов;

Задание #2
Напишите программу, которая анализирует массив студентов и определяет, какие оценки необходимо получить,
чтобы стать лучше 25% студентов, 50% студентов, 75% студентов.

Задание #3
Напишите программу которая находит агента с наивысшым медианным значением по месяцам. (Количество агентов 10)
struct Agent
{
  int monthlySales[12];
}
Месяца, в которые агенты не работали в массиве продаж помечаются -1.
Диапазон значений зарплат от 100 до 1000
Медиана, это так называемое значение "посередине". То есть такое значение,
что половина других значений меньше него, половина других - больше.
Если значений четное число, то медиана - это среднее значение двух значение "посередине"
Например, если имеется следующий набор данных: 10, 6, 2, 14, 7, 9, то два значения "посередине"
это 7 и 9. Среднее значение между 7 и 9 это 8 и есть медиана данного набора данных.
Необходимо учитывать, что агенты могут быть наняты или уволены в середине года.
